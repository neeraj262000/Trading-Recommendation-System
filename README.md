# Trading Recommendation System

This project uses a Convolutional Neural Network (CNN) to detect **Double Top** and **Double Bottom** chart patterns in stock market screenshots. Based on the detected pattern, the system provides trading recommendations: **Buy** for Double Bottom and **Sell** for Double Top.

## Features

- Automated detection of Double Top and Double Bottom patterns
- Image-based input for stock charts
- CNN-powered classification for reliable pattern recognition
- Actionable trading recommendations

## Getting Started

### Prerequisites

- Python 3.8+
- TensorFlow/Keras
- OpenCV
- NumPy
- Pandas

### Installation

1. Clone the repository:
git clone https://github.com/neeraj262000/trading-recommendation-system.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the Jupyter notebook:
jupyter notebook CNN-Trading-Recommendation-based-on-Chart-Patterns.ipynb


## Usage

- Place stock chart screenshots in the designated input folder.
- Run the notebook to process images and receive trading recommendations.
